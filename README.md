# Falkor

Named after the luck dragon from _The Never-Ending Story_, Falkor is a set of development tools for working with DragonRuby.

As of right now, they're just a set of things I've written to learn about DragonRuby development and solely for my own use. That could change in the future, but for now these will be created with my own development proclivities in mind and not necessarily what will be most useful to most ruby game development.
